Detailed Description

Interface for FormRepository. This interface is required to support the use of dependency injection. Dependecy injection allows the creation of Controller constructors that accept this interface. At runtime the constructors are intercepted and properly mapped to the repositories that implement this interface. The benificial side effect, and reason for this, is that this opens the controllers up for unit testing by stubbing/mocking.
